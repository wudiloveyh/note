# 终端美化

## 下载nerd字体

https://www.nerdfonts.com/font-downloads

因为有些字符在终端显示不出来，需要下载字体才能显示

## 安装oh-my-posh

powershell运行：

```powershell
winget install oh-my-posh
```

创建配置文件：

```powershell
new-item -type file -path $profile -force
```

修改配置文件：

```powershell
notepad $PROFILE
```

添加以下内容：

```powershell
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\paradox.omp.json" | Invoke-Expression
```

## 修改主题

其中的paradox就是主题名字，主题展示在下面的网站中：

[Themes | Oh My Posh](https://ohmyposh.dev/docs/themes#zash)

修改对应的主题名字就可以了
